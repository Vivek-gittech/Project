# Cafe UI - React (JavaScript) + Vite + Tailwind

## Quick start
1. unzip the project
2. run `npm install`
3. run `npm run dev`

Make sure you have a backend that serves API endpoints at `http://localhost:8080/api/menu/get`
and `http://localhost:8080/api/menu/category/:category`. If you don't have one, run a simple mock server.
